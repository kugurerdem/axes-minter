## Setup

Create a .env file and fill the following data.
  ```
  PRIVATE_KEY=<private_key> 
  RPC_URL=https://bsc-dataseed.binance.org/ 
  CHAIN_ID=56 
  ```

You need npm installed. Then run the following code
  ```
  npm install
  ```
To run the bot you can type
  ```
  node index
  ```
